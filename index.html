PROJECTS

Wall-I Smart Dustbin

Working towards a sustainable future, one trashcan at a time

A smart dustbin that identifies an object and then segregates it into the right category. This project went on to win UofTHacks 6, Toronto's Largest Hackathon

I used Azure Custom Vision and IBM Watson to analyze the waste and categorize it. Using an Arduino connected via Azure IoT Hub I made the dustbin functional for the right segregation of the waste.

Huffman Lossless Compression

Reducing file size, for any file

I used a binary search tree to construct a lossless compression algorithm that can reduce the size of any file. This algorithm has the power to reduce a file to 1/5th of its original size.

Training AI to play Flappy Bird using NEAT

seeing AI playing fappy bird

I used the distance between the bird and the top tube alongside the distance between the bird and the bottom tube to train the model. The model is so advanced, that only after 1 set of training, the bird can continue flapping forever, ie: we have beaten the game.

Visualizing nearest library in NYC

Visualizing the map of NYC based on the distance to the nearest library

I used the public dataset of all the libraries in manhattan and then calculated the distance from every building in manhattan to the nearest library. Using this distance, I mapped NYC colour coded on the distance to the nearest library
